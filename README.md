# 👑 SQLPhone Emperor

**Database mastery from scratch, built entirely on an Android phone.**  
SQLite + Termux. 12 modules. 98 lessons. The second pillar of the full‑stack empire.

---

## 🛠️ Tools
- [Termux](https://termux.com/) – terminal emulator
- SQLite (pre‑installed in Termux via `pkg install sqlite`)
- Python 3 + `sqlite3` (for integration modules)
- [Acode](https://acode.foxdebug.com/) – code editor with Markdown preview
