# ♻️ Spaced Repetition – Review Modules 01–03

## 🧪 Task
1. Create a table `sales` with columns: `id INTEGER PRIMARY KEY`, `product TEXT`, `category TEXT`, `amount REAL`, `sale_date TEXT`.
2. Insert at least 6 rows, covering multiple categories and dates.
3. Write a query that shows total `amount` sold per category.
4. Filter the above to show only categories with total > 100.
5. Write a query that finds the earliest and latest sale dates.
6. Using a subquery (or join), show products whose amount is above the overall average.
