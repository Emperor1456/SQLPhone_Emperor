# ♻️ Spaced Repetition – Review Modules 01–06

## 🧪 Task
1. Create a table `events` with `id`, `name`, `event_date TEXT`.
2. Insert events with dates covering at least 3 different months.
3. Use `strftime()` to extract the month and show count of events per month.
4. Concatenate `name` with the day of the event (`|| ' - ' || day`).
5. Use `COALESCE` to display 'No Event' if name is NULL (insert one NULL row).
6. Cast a date string to a different format using `strftime()`.
