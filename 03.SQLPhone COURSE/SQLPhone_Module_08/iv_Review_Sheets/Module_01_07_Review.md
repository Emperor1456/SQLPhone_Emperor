# ♻️ Spaced Repetition – Review Modules 01–07

## 🧪 Task
1. Create a table `students` with `id`, `name`, `score`.
2. Insert at least 5 students with scores between 40 and 100.
3. Use a `CASE` expression to display a grade (A, B, C, F).
4. Create a view `top_students` that shows only students with score >= 80.
5. Query the view.
6. Export the view's result to a CSV file using SQLite CLI commands (write them as a comment).
