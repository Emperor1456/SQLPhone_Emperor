# ♻️ Spaced Repetition – Review Modules 01–02

## 🧪 Task
1. Create a table `inventory` with columns: `id INTEGER PRIMARY KEY`, `item TEXT NOT NULL`, `quantity INTEGER`, `warehouse TEXT`.
2. Insert at least 5 items, some with the same warehouse.
3. Write a query that shows all unique warehouses (`SELECT DISTINCT`).
4. Write a query that shows items with `quantity < 10` and belonging to `'A'` warehouse.
5. Sort the results by `quantity` descending.
6. Add a comment explaining the second query.
