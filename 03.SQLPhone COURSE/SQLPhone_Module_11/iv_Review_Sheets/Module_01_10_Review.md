# ♻️ Spaced Repetition – Review Modules 01–10

## 🧪 Task
Design a mini‑project schema for a **recipe manager**:
- `recipes` (id, name, instructions)
- `ingredients` (id, name)
- `recipe_ingredients` (recipe_id, ingredient_id, quantity)
Create the tables, insert sample data (3 recipes, 5 ingredients), and write queries:
1. List all recipes with their ingredients (use JOINs).
2. Find recipes that contain a specific ingredient.
3. Count the number of recipes each ingredient appears in.
4. Identify ingredients not used in any recipe (LEFT JOIN + NULL).
5. Add constraints and indexes as appropriate.
