# ♻️ Spaced Repetition – Review Modules 01–09

## 🧪 Task
1. Create a table `users` with columns: id, username (UNIQUE), password, email.
2. Insert at least 3 rows.
3. Use `EXPLAIN QUERY PLAN` on a query that searches by username.
4. Create an index on `username` and repeat the `EXPLAIN QUERY PLAN`.
5. Simulate a SQL injection attack (in your mind) and explain in a comment how parameterized queries prevent it.
6. Write a transaction that transfers 100 from one user (in a dummy `accounts` table) to another, rolling back if insufficient funds.
