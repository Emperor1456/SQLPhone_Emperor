# 📝 Module 11 – Real‑World Projects

## Projects Overview
1. Student Management System (many‑to‑many via enrollment)
2. E‑commerce Inventory (categories, suppliers, sales)
3. Library Management (books, members, loans)
4. Employee Payroll (departments, salary history)
5. Blog Database (users, posts, comments)
6. Expense Tracker (categories, time‑based aggregation)
7. Movie Rating System (many‑to‑many ratings)
8. Custom Project (your own design)

## Common Patterns
- Junction tables for many‑to‑many.
- Foreign keys with cascading actions.
- Date/time aggregation with `strftime()`.
- Business reports using `GROUP BY`, `HAVING`, and `JOIN`.

## Deliverables for Each
- DDL with constraints.
- Sample data (min 5 rows per table).
- Set of 5 meaningful queries (joins, aggregation, filtering).
